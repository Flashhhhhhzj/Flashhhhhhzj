## 🔧 About My Skills

These repositories represent three different *skills* or high‑impact workflows I’ve worked on:

- 🔁 **skill‑runoob2md (Private)** — A skill to pull Runoob tutorial articles *verbatim* into a Markdown knowledge repo.  
- 📄 **feishu‑docx‑to‑markdown (Public)** — Converts Feishu/XFChat cloud document links to Markdown, preserving headings, numbers, tables, code blocks, images and links.  
- 🧠 **care‑course‑import‑poc (Private)** — Proof of concept for recursively parsing Feishu wiki content and using an LLM to extract course info for pre‑filled lesson forms. This logic is being migrated into the `care‑dev` project.
